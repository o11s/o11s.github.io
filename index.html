open80211s

What is it?

open80211s is an open-source implementation of the IEEE 802.11 wireless mesh standard. The goal is that the software runs any commodity hardware supported by the Linux kernel.

Goals

  • To create the first open implementation of 802.11s.
  • To let the world use it, understand it and contribute to it.
  • To consolidate the numerous non-interoperable mesh protocols into one that is based on a standard specification.
